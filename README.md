# Realm Ruckus Web

This repository contains the standalone Realm Ruckus website.

Scope:

- website frontend and static assets;
- online rules and card presentation;
- SEO, deployment configuration and website tests;
- explicit content synchronization and dependency locks.

Website deployment must bind only to this repository. Core, Production and Theme commits must not automatically deploy the website.

Current migration source: `anxiannet/realmruckus`.
